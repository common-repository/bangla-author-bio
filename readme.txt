== Description ==

Simply adds a author bio after every post automatically in Bangla. You can integrate Twitter and Facebook profiles.

== Installation ==

Step 1. Download the Plugin from http://wp-adm.in

Step 2. Upload it in you wp-content/plugins dirrectory.

Step 3. Activate from Admin panel.

Step 4. Write a proper author bio for every author of the blog.
